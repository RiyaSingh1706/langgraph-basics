# LangGraph Learning

This repository contains my hands-on practice with LangGraph. I'm using it to understand how agent workflows are built by implementing one concept at a time, starting from the basics and gradually moving to more advanced topics.

## Topics Covered

* Basic state graphs
* State updates
* Multi-node workflows
* Reducers
* Conditional routing
* Loops
* Checkpointing and memory

More concepts will be added as I continue learning, including tool calling, LLM integration, RAG, and multi-agent workflows.

## Getting Started

```bash
git clone <repository-url>
cd Langgraph

python -m venv .venv
```

Activate the virtual environment and install the required packages:

```bash
pip install langgraph langchain typing_extensions
```

Run any example:

```bash
python step-1.py
```

## Purpose

The goal of this repository is to document my learning process and build a solid understanding of LangGraph before using it in larger AI projects.
